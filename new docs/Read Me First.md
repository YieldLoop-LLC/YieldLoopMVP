## What YieldLoop Actually Is

YieldLoop is an **automated trading system** built around one simple idea:

> Your money should never depend on someone else behaving well.

When you use YieldLoop:
- You don’t deposit into a pool
- You don’t backstop other users
- You don’t prop up a token

You deposit into **your own vault**.

One vault.  
One owner.  
One outcome.

## The Ratcheting Floor (Why LOOP Is Different)

This is the part degens notice — and skeptics should.

LOOP has a **ratcheting floor**.

When the system earns real profit:
- A portion is allocated to a reserve
- That reserve backs LOOP
- Once the backing increases, it does **not** decrease

The floor only moves one way.

No profit?  
No movement.

Profit?  
The floor steps up — permanently.

Slow.  
Deliberate.  
Unexciting.

That’s the point.
